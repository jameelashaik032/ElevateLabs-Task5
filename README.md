# ElevateLabs-Task5
Decision Trees and Random Forests

## Objective
Learn tree-based models for classification.

## Dataset
Titanic Dataset

## Steps Performed
- Loaded Titanic dataset
- Removed unnecessary columns
- Converted categorical values into numerical values
- Handled missing values
- Split data into training and testing sets
- Trained Decision Tree Classifier
- Visualized Decision Tree
- Controlled overfitting using max_depth
- Trained Random Forest Classifier
- Compared model accuracies
- Analyzed feature importance
- Performed 5-Fold Cross Validation

## Results

Decision Tree Accuracy: 77.65%

Random Forest Accuracy: 82.12%

Cross Validation Average Score: 81.15%

## Conclusion

Random Forest performed better than Decision Tree and achieved higher accuracy.
